1. Метод join(). Метод масиву join(delimiter) дозволяє об'єднати елементи масиву в рядок.
2. Метод split(). Метод рядків split(delimiter) виконує зворотну операцію порівняно з методом масивів join(delimiter). Він дозволяє перетворити рядок на масив, розбивши його за вказаним роздільником delimiter.
3. Метод slice(). Метод масиву slice(begin, end) повертає новий масив, що містить копію частини вихідного масиву, не змінюючи його. Копія створюється від індексу begin до, але не включаючи індекс end.
4. Метод concat(). Метод concat(arr1, arr2, ..., arrN) використовується для об'єднання двох або більше масивів.
5. Метод indexOf(). Метод масиву indexOf(elem) використовується для визначення індексу першого входження елемента elem у масиві. Він повертає індекс елемента, якщо він знайдений, або -1, якщо елемент не знайдений. Метод indexOf() виконує сувору рівність (===) при порівнянні елементів.
6. Метод push(). Метод масиву push() використовується для додавання одного або більше елементів у кінець масиву.
7. Метод Array.from(). Метод який створить масив із псевдомасиву.
